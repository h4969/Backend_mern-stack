# Vendor Dashboard - Backend

This is the **backend API** for the Vendor Dashboard project, built with **Node.js and Express.js**.  
It provides secure **RESTful APIs** for vendor authentication, firm management, and product management.  

---

## 🚀 Features
- **Vendor Authentication** – Registration & Login using **JWT**.
- **Firm Management** – Create and manage firm details.
- **Product Management** – CRUD operations with image upload support.
- **Secure Routes** – Middleware for token validation.
- **Error Handling** – Standardized responses for client errors.

---

## 🛠️ Tech Stack
- **Node.js** (runtime)
- **Express.js** (framework)
- **MongoDB & Mongoose** (database & ORM)
- **Multer** (file upload)
- **JWT (JSON Web Token)** (authentication)
- **dotenv** (environment config)
